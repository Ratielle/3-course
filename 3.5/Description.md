# Интерполяция одномерной функции
- Кусочно-линейная интерполяция
- Интерполяция кубическими сплайнами с заданными вторыми производными на концах 
# HowTo:
- Проверено на MSVC компиляторе
- Для сборки выполнить в консоли команды
  - `cmake -G "Visual Studio 16 2019" -B ./build .`
  - `cmake --build ./build`
- Для запуска выполнить команду `./output/Debug/plotter.exe [options]`

  Список параметров:
  - -LB, --left_boundary - левая граница области определения
  - -RB, --right_boundary - правая граница области определения
  - -NP, --num_plot_points - число точек для отрисовки
  - -NI, --num_inter_points - число сегментов для интерполяции
  - -C, --cubic - использовать кубическую интерполяция вметсо линейной
  - -LD, --left_derivative - значение второй производнйо на левом конце
  - -RD, --right_derivative - значение второй производной на правом конце

  Пример использования:
  `./output/Debug/plotter.exe -LB=-5 -RB=2 -NP=1000 -NI=10 -C -LD=2 -RD=-5`
